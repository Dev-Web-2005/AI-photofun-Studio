Introduction to commit github

feat: thêm tính năng mới

fix: sửa lỗi

chore: các tác vụ “vệ sinh”, không ảnh hưởng logic app (ví dụ config, scripts, build)

refactor: thay đổi code structure, không thêm feature, không fix bug

docs: thay đổi tài liệu

style: format code, whitespace, không ảnh hưởng logic

perf: tối ưu performance

test: thêm/sửa test
